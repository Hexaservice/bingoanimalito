# Carpeta de sonidos (política local)

Esta carpeta contiene **todas** las fuentes de audio permitidas por la app.

## Regla obligatoria
- Solo se aceptan rutas locales del repositorio:
  - `/sonidos/*.wav`
  - `sonidos/*.wav`
- No se permiten URLs externas (`http://`, `https://`, `//`).
- El motor de audio ignora y rechaza fuentes fuera de `public/sonidos`.

## Archivos actuales
- `1.wav` a `75.wav` (efectos y cantos numéricos).

## Recomendaciones técnicas
- Formato preferido: `wav`.
- Mantener duración corta para SFX (< 2s).
- Nivelar loudness en origen para consistencia de volumen.
